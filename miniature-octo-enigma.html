Úvod

Jako téma semestrálního projektu jsem si vybral xml pro nastavení serveru pro fiktiví počítačovou hru

XML soubor slouží k obecné konfiguraci serveru, také se v něm shromaždují informace o score připojených hráčů a informace o jejich nastavení hry aby se předešlo podvodům s upravou nastavení.

Metodika

Využité programy

Pro tvorbu tohoto projektu jsem použil programy Visual Studio Code a prohlížeč Google Chrome.

Využité webové stránky

Většinu informací jsem zjískal na stránkách W3Schools, všechny validace spojené s xml jsem použil stránky utilities-online.info, konkrétně pro xml a xsd a pro xslt a pro html jsem použil oficální w3 schools validátor
